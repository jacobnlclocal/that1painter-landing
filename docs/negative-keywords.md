# Negative Keyword List — That 1 Painter Knoxville

Built from the audited Search Terms report (Mar 29 – May 12, 2026).

in the audit window.

## How to add to Google Ads

1. **Tools → Shared Library → Negative keyword lists** → **+**
2. Name it: `T1P Knoxville - Master Negatives`
3. Paste each themed block below into "Add multiple keywords"
4. Hit Save
5. Attach to **every Search campaign** in the account

Each line is one keyword.
- `[keyword]` = Exact match negative — blocks only that exact query
- `"keyword"` = Phrase match negative — blocks any query containing the phrase
- Bare keyword = Broad match negative — AVOID unless explicitly noted

> **Rule:** never broad-match a negative unless you understand exactly what it blocks.
> Broad negatives have killed more painter accounts than they've helped.

## What this list does NOT include

- **Brand keywords (`that 1 painter`, `that1painter`)** — these are YOUR brand and convert at $67/lead. Leave them positive.
- **Generic "painters near me" variants** — these convert at higher CPA but they ARE buyers. Don't block; just structure with Smart Bidding.
- **Geographic suburbs you serve (Farragut, Maryville, Powell, Oak Ridge, Sevierville)** — leave positive.
